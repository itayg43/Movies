# Movies

## Overview

Mobile app built using TypeScript, React Native (Native CLI), React Navigation, and Redux.
The app allows users to scroll and search for movie data using TMDB API.
When tapping on the movie card, the user will be navigated to a specific page to get more information, including a YouTube trailer link and a list of recommended movies.
